# FaBoGPIO-PCAL6408-Python

## How to install.

```
# python 2.7
pip install FaBoGPIO_PCAL6408
# python 3.6
# pip3 install smbus2
pip3 install FaBoGPIO_PCAL6408
```

## FaBo GPIO I2C Brick

　[#210 GPIO I2C Brick](http://fabo.io/210.html)

## PCAL6408A

　PCAL6408A is 8bit I2C I/O expander.
